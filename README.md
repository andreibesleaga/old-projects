- alpha-laboratory-proiect_final.zip - Alpha Remote Virtual Laboratory - LabVIEW & Internet Toolkit - world first implemented & used remote virtual laboratory of connected electronics hardware for remote learning in electronics universities (~ 2004)
- zfeeder-1.6.zip - zFeeder - world's first PHP opensource RSS Aggregator with admin backend, PHP5 (~ 2004)
- zshaped-1.2.zip - newssystem/blog, PHP4 learning project (~ 2003)
