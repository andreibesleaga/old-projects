# Historical innovative unmaintained archived projects

Each project has own copyright/license.

- Bucharest.Source-v2.0.1.0.zip - Bucharest's first Info Portal Mobile App - Windows Phone, App Studio (~ 2010)
  
- phspectrum-0.1.zip - phSpectrum - ZXSpectrum 8bit Emulator in PHP/PHP-GTK, proof of concept - world's first hardware computer emulator in PHP, running code from the past (~ 2008)
  
- alpha-laboratory-proiect_final.zip - Alpha Remote Virtual Laboratory - LabVIEW & Internet Toolkit - world first implemented & used remote virtual laboratory of connected hardware electronics, remote e-Learning platform in electronics universities (~ 2004)
  
- zfeeder-1.6.zip - zFeeder - world's first PHP opensource RSS Aggregator with admin backend, PHP5 (~ 2004)
  
- zshaped-1.2.zip - newssystem/blog, PHP4 learning project (~ 2003)


Other open-source participations before GitHub era, included:
- ADODB PHP SQLite drivers
  
- Apache Web Server 2.0 (https://httpd.apache.org/docs-project/contributors.html)\
  
- SmallBASIC demo programs sets
  
- other programs & contribs
